Misión

Optimizar la gestión de suministros en empresas comerciales mediante el desarrollo de un sistema automatizado de monitoreo que integre bases de datos SQL y lógica de programación, garantizando que nunca exista una ruptura de stock y eliminando el error humano en el registro de inventarios.

Visión

Convertirse en la solución estándar de automatización de inventarios para pequeñas y medianas empresas (PyMEs), transformando la gestión logística manual en un proceso inteligente, predictivo y basado en datos que permita el crecimiento escalable de los negocios.

Objetivo

Diseño de Base de Datos: Estructurar una base de datos relacional en SQL que registre productos, proveedores y movimientos de almacén en tiempo real. Automatización de Alertas: Programar un script (en Python o JavaScript) que consulte la base de datos cada cierto tiempo y envíe notificaciones automáticas cuando el stock sea bajo.
